## 2. The Classical Foundation: Doppler as Rate Modulation

### 2.1 The Universal Rate Equation

**Theorem 2.1** (Universal Doppler Principle): Any periodic process exhibits Doppler-like behavior when source and observer have relative motion.

**Proof**:
Consider a source emitting pulses at rate R_source. Each pulse travels at speed c_wave. With relative velocity v:

Time between pulse emissions (source frame):
```
Δt_source = 1/R_source
```

Distance between pulses:
```
Δx = c_wave × Δt_source = c_wave/R_source
```

For approaching motion, the observer encounters pulses faster:
```
R_observed = c_wave/(Δx - v×Δt_source)
          = c_wave/[(c_wave - v)/R_source]
          = R_source × c_wave/(c_wave - v)
```

This is the universal Doppler formula. □

### 2.2 Examples Across Physics

**Sound Waves** (ambulance):
```
f_obs = f_source × c_sound/(c_sound ± v_source)
```

**Light Waves** (stars):
```
f_obs = f_source × c/(c ± v_star)  [classical approximation]
```

**Water Waves** (boat wakes):
```
f_obs = f_source × c_water/(c_water ± v_boat)
```

**Key Insight**: Doppler is about counting rates, not specific to any wave type.

### 2.3 The Relativistic Correction

When wave speed equals c (invariant), both source and observer time dilate:


This is the relativistic Doppler formula!
